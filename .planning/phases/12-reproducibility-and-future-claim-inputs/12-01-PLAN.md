<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Raw JSON artifact -> Phase 12 source loader | Local experiment artifacts may be missing, malformed, stale, or non-PASSED. |
| Source statuses -> claim inputs | Statuses determine whether rows are claim-eligible, context-only, limitations, or failed. |
| Generated text/CSV/JSON -> future manuscript input surface | Overclaiming language could leak into later paper work if not scanned. |
| CLI paths -> local filesystem | Caller-provided or default paths control local reads/writes. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-01 | Tampering | `load_source_records` | mitigate | Record exists/parse/source status for each required registry entry and never silently skip missing or invalid JSON. |
| T-12-02 | Repudiation | provenance manifest builders | mitigate | Emit source paths, statuses, commands, requirements, generated artifacts, and caveats for every derived output. |
| T-12-03 | Information Integrity | claim-input builders | mitigate | Preserve simulator/network/horizon/warmup/seed/profile/gate qualifiers and set `claim_allowed=false` for SMOKE_ONLY, PILOT_ONLY, INCONCLUSIVE, and unsupported dominance categories. |
| T-12-04 | Information Integrity | claim audit | mitigate | Run `claim_policy.forbidden_claim_hits` plus Phase-12-specific forbidden phrases across generated JSON/CSV/Markdown surfaces and fail strict mode on hits. |
| T-12-05 | Denial of Service | reproduction manifest | mitigate | Default commands package status and regeneration; long-horizon SUMO execution is opt-in/fail-closed and not invoked by default generation. |
| T-12-SC | Tampering | package installs | accept | No package-manager install tasks exist; implementation must use Python stdlib and existing project scripts only. |
</threat_model>
