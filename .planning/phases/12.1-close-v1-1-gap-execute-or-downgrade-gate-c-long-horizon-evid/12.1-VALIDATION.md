## Sampling Rate

- **After every code-change task:** Run `python tests/test_phase11_paired_evidence.py` for Phase 11 runner/Gate C changes.
- **After Phase 12 propagation changes:** Run `python tests/test_phase12_reproducibility_inputs.py`.
- **After every execution batch/checkpoint:** Inspect Phase 11 artifact/checkpoint row counts, missing row keys, and fail-closed status.
- **Before `/gsd:verify-work`:** Run the full relevant suite, regenerate Gate C, refresh Phase 12 artifacts, and inspect final status/row-count invariants.
- **Max feedback latency:** Fast code/test feedback should be seconds; long-horizon execution feedback is batch/checkpoint bounded.
