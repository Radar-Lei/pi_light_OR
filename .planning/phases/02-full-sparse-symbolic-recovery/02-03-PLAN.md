## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Solver payload → JSON/CSV/TXT artifacts | Internal optimization results become auditable files consumed by later phases. |
| Rule renderer → human interpretation | Rendered text may be mistaken for executable or closed-loop validated policy. |
| Output paths → filesystem | CLI paths determine which experiment artifacts are overwritten. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-08 | Elevation of Privilege | Rule text renderer | mitigate | Render plain text only; never `exec`, import, or deploy generated rules in Phase 2. |
| T-02-09 | Repudiation | Phase completion status | mitigate | Gate `PASSED` on schema/output/K>1 solve completeness and include boolean gates in JSON. |
